Projects

A group project focused on analyzing the lyrical themes and stylistic patterns of Olivia Rodrigo in comparison to other influential female pop artists in the same musical genre. By utilizing Python Natural Language Processing (NLP) techniques and visualization methods, our group was able to uncover distinctive patterns and themes throughout Olivia Rodrigo's songwriting.

A group project focused on encoding and analyzing a volume of Dennis the Menace using Comic Book Markup Language (CBML). Each page was encoded into XML format while adhering to the CBML Relax-NG guidelines. Additionally, our group developed a custom XSLT to transform the XML data into HTML for webpage viewing.

An interactive 3D rotating dessert platter scene using Three.js. Users can click on a dessert model to rotate the platter and learn more about each dessert. The dessert models were made in Blender and loaded into the scene using GLTFLoader. Raycaster is used to detect hover and click interactions.

A detailed analysis of the game Style Savvy: Trendsetters for the Nintendo 3DS focused on gameplay, characters, challenges, and mechanics. Includes a Kumu Network which visualizes and describes certain game aspects such as clothing brands, locations, etc.
